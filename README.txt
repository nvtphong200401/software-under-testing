Cách vận hành trang web:
* Cách 1: Sử dụng link web đã up lên host 
- Truy cập vào trang web: https://auctionx.herokuapp.com/

* Cách 2: Chạy chương trình trên localhost
- Chạy file auction.sql ở trong folder queries để khởi tạo db (db mà nhóm sử dụng là MySQL)
- Mở file utils/db.js để sửa lại thông tin database cho phù hợp với máy tính
- Mở cmd sử dụng câu lệnh "node app" để chạy chương trình
- Sử dụng trình duyệt web bất kì, nhập localhost:3000 để xem được web
